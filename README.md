# Automated Stock Scraper
Using the online brokerage services of **Charles Schwab**, this python script scrapes the following variables for a highlighted stock:
### Greek Values
+ Vega
+ Theta
+ Gamma
+ Delta
+ IV
+ Bid
+ Ask 
+ Strike
+ Bid2
+ Ask2
+ IV2
+ Delta2
+ Gamma2
+ Theta2
+ Vega2

### Call & Put Values
+ Volume
+ OI
+ Last
+ Change
+ Bid
+ Ask
+ Strike
+ Bid2
+ Ask2
+ Last2
+ Change2
+ Volume2
+ OI2
